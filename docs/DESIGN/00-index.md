## Design Principles

1. **Royal/Industrial** — Premium laundry service with a technical, precise edge
2. **Bold & Clear** — Heavy typography, high contrast, generous whitespace
3. **Status-Driven** — Production statuses are the core UX (dots, badges, progress bars)
4. **Dark Mode Ready** — All components support light/dark toggle via `.dark` class
5. **Mobile-First Admin** — Responsive sidebar, collapsible navigation, touch-friendly
